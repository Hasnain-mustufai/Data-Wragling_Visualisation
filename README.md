# Data-Wragling_Visualisation
Projet : création d’un manuel complet couvrant 95 % des cas essentiels en Data Wrangling Python(Pandas). Standardisation, nettoyage, transformation des colonnes et valeurs, gestion des outliers, typages, imputations, regex, et visualisations clés (Seaborn, Matplotlib) (bar, pie, scatter, heatmap, boxplot).
